💻 Projeto: Página de ponto turístico - Rocketseat
Este é um projeto prático desenvolvido durante o curso da Rocketseat, do ciclo Riseup pelo Porto Digital. O objetivo foi criar uma página informativa sobre pontos turísticos de uma cidade. Decidi reproduzir o exemplo da cidade de Busan, focando no domínio de HTML e CSS.

🚀 O que foi praticado:
HTML Semântico: Organização clara usando header, main, section e footer.

CSS Moderno: Uso de Flexbox para alinhamento e bordas arredondadas em imagens.

Layout Fiel: Estilização baseada em um design proposto, cuidando de cores e fontes.
